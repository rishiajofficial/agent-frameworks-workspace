# AutoGen example

Minimal conversational setup: AssistantAgent and UserProxyAgent. Good for chat-driven workflows and code execution.

## Setup

Copy `../.env.example` to `.env` and set `OPENAI_API_KEY`. Then:

```bash
uv sync
```

## Run

```bash
uv run python main.py
```

Docs: [AutoGen](https://microsoft.github.io/autogen/)
